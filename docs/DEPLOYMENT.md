# Deployment Guide

Deploy **MAJ Boutique** to Vercel (recommended, free tier).

## Steps

1. Import [Prototype-MAJ](https://github.com/ztajwer/Prototype-MAJ) on [vercel.com](https://vercel.com)
2. Set environment variable: `NEXT_PUBLIC_SITE_URL=https://your-app.vercel.app`
3. Deploy

See full steps in repository README.

## Requirements

- Node.js 18+
- Do not upload `node_modules` or `.next`
